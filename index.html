How to Use This Website

This tool helps supervisors find when multiple students are available by analyzing their weekly schedules.

  1. Students must first go to Gizmo and export or screenshot their weekly schedule.
  2. Go to the Upload page and upload the schedule image.
  3. Enter the student's name and confirm the start time of the schedule.
  4. Click Add Schedule to save it.
  5. Repeat this for every student whose schedule you want to analyze.
  6. Go to the Query page to find shared availability across all saved schedules.

Schedules are stored locally in your browser, so you only need to upload them once.
